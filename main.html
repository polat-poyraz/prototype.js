Prototype.js
Prototype.js ile prototiplerinizi daha rahat yazın.
başlatmak için terminalden prototype.js dosyasına girin ve şunu yapın;
> npm start
prototypes klasör mimarisi şu şekildedir
  • public
    • first.html
    • last.html
  • styles
    • index.css
css dosyalarınızı style içine html dosyalarınızıda public in içine koyunuz.
bir prototipe girmek isterseniz localhost:800/prototipname şeklinde yazın
warning
Eğer public klasörüne bir protitp eklerseniz (html dosyası)
ozaman terminali ctrl + c ile durdurup tekrar başlatmalısınız.
warning
gereksinimler için kullanmadan önce terminalde
> npm install
yapmayı unutmayın
