Prototype.js

Başlangıç

başlatmak için terminalden prototype.js dosyasına girin ve şunu yapın;
> npm start
prototypes klasör mimarisi şu şekildedir
  • assets
    • sample.jpg
  • components
    • first-component.js
  • public
    • first.html
    • last.html
  • styles
    • index.css
bir prototipe girmek isterseniz localhost:8000/prototypename şeklinde yazın
warning
Eğer public klasörüne bir prototype eklerseniz (html dosyası)
terminali ctrl + c ile durdurup, npm start ile tekrar başlatmalısınız.

Component

prototype.js size statik componentler yapmanızı sağlar. Statik componentler javascript açısından kullanışlı değildir yanlızca dinamik html ve css componentleri yazılmalıdır.

Component kullanmak istediğiniz dosyaya
<script src='../lib/prototypeComponent.js'></script>
<script src='../components/kullanılacak-bileşen.js'></script>

kullanmak istediğimiz dosyaya prototypeComponent.js i ekledik.
artık bir component yazıp bunu kullanabiliriz.
Component yazmak için prototypeComponent.js den gelen componentUse() fonksiyonunu kullancağız.
componentUse() fonksiyonu içerisine bir callback function ve bir id adresi alır. içerisindeki callback fonksiyon bir html string i return eder. örneğin;
componentUse(() => {
const title = 'First Component'
// Her componentsUse kullanımında bir callback fonksiyon html kodlarını return etmelidir.
// html kodları return edilirken `` tırnak kullanılmalıdır.

return (`
<h1 class="first-component"> ${ title } </h1>
<style>
.first-component{
background: black;
color: white;
text-align: center;
padding: 1em;
margin-bottom: 2em;
}
</style>
`)
}, '#first-component' /* Componenti hangi isim ile kullanılacağı yazılır. eğer id ise # class ise . olarak türü belirtilmelidir*/)

şimdi basit bir component yazdık bu component içerisinde html ve css kodları bulunduruyor ve bir id alıyor. Bu component i kullanabilmek için component i bağladığınız html dosyasına componentUse() fonksiyonuna verdiğiniz id ile bir etiket açmanız yeterlidir. örneğin;
<div id="first-component"></div>
Componentimizi yazdık ve kullandık. prototype.js in özellikleri bu kadardır.
Hızlı, rahat ve düzenli prototype yazmanızı sağlar.
